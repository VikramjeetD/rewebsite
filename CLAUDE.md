# Project: Brandy Culp Realty Website

## Build Rules
- Do NOT run `npm run build` after every single change. Only build when explicitly asked or when all changes for a task are complete and need verification.
- Batch multiple related edits before verifying with a build.

## Stack
- Next.js 15 (App Router), React, TypeScript, Tailwind CSS
- Firebase/Firestore for data
- Deployed on Vercel
